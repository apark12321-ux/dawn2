# DAWN 집대성 설계도 — 흩어진 주식 도구를 하나로

> 1년간 만든 주식 도구는 전부 "한 조각"이었다.
> DAWN이 그 조각들을 끼워 맞추는 **하나의 집**이 된다.

---

## 1. 자산 전체 (네가 만든 것)

| 도구 | 한 일 | 핵심 로직 |
|---|---|---|
| 종목 스크리너 (KIS, Python 4파일) | 코스피·코스닥 일일 후보 선별 | 5일 OHLCV 신호 + 과열제외 + 비용모델 |
| 4-Arrow 수렴 분석기 (React) | 4지표 정렬 + 수급 에너지 점수 | DMI·MACD·스토캐스틱·CCI + 에너지 5조건 + 한줄 판정 |
| 삼성전자 매물대 (KIS WebSocket) | 실시간 체결 → 가격대별 거래량 | H0STCNT0 틱 누적 → POC |
| 비용 모델 | 단타 실제 기대값 | 왕복비용(수수료·NXT·세금·슬리피지) → 순+1% = 총+1.46% |
| 계좌 담보·상환 시뮬레이터 (Excel) | 담보비율·반대매매·매도우선순위 | 140% 유지 시나리오 |
| DAWN / 여명 (현재) | 아침 브리핑 소비자 앱 | 거시→수급→주목주 퍼널 |

**공통 척추**: 매수/매도 신호는 안 준다. 관찰·근거만. (신뢰 + 유사투자자문업 회피)

## 2. 통합 설계 — DAWN 4겹

```
┌─────────────────────────────────────────────┐
│  얼굴 FACE   DAWN 브리핑 UI (이미 완성)          │
├─────────────────────────────────────────────┤
│  두뇌 BRAIN  스크리너 점수엔진                    │
│             주목주를 '거래대금 순위'가 아니라      │
│             '신호 점수'로 선별 + 과열제외          │
├─────────────────────────────────────────────┤
│  깊이 DEPTH  종목 탭 → 4-Arrow 수렴 + 에너지 점수 │
│             + 한줄 판정 + 매물대                  │
├─────────────────────────────────────────────┤
│  해자 MOAT   키움 실시간 체결(0B) → 매물대         │
│             (고정IP 워커, 못 베끼는 데이터)        │
├─────────────────────────────────────────────┤
│  척추 SPINE  관찰 원칙 + 비용 정직성               │
└─────────────────────────────────────────────┘
```

## 3. 무엇을 어디에 꽂나

| 자산 | → DAWN 위치 | 상태 |
|---|---|---|
| 스크리너 신호엔진 | `/api/briefing` 주목주 선별 (ka10032 단순순위 → 점수랭킹) | `src/lib/signals.ts` 이식 완료 |
| 4-Arrow + 에너지 | 종목 상세 모달 (일봉으로 계산) | `src/lib/indicators.ts` 이식 완료 |
| 비용 모델 | 종목 카드/상세 "정직" 한 줄 | signals.ts 포함 |
| 매물대 (WebSocket) | 종목 상세 매물대 | 고정IP 워커 필요(키움 지정단말기) |
| 담보 시뮬레이터 | (선택) PRO 리스크 관리 탭 | 보류 |

## 4. 빌드 순서 (큰 것부터)

1. **두뇌**: 스크리너 점수엔진 → `/api/briefing`에서 일봉(키움) 받아 주목주를 진짜 '선별'. 각 종목 "왜?"에 신호 근거 자동 문구.
2. **깊이**: 종목 상세에 4-Arrow(n/4) + 에너지(5조건) + 한줄 판정.
3. **해자**: 고정IP 워커 + 키움 실시간 체결 매물대.
4. **정직**: 비용모델 한 줄 — "이 종목 +x%지만 단타 순수익선은 +1.46% 필요".

---

## 5. 지금 블로커 (정직하게)

- 키움 **실전** = 지정단말기(8050) → 고정IP 워커 또는 해제 필요
- 키움 **모의** = 순위·실시간 데이터 제한될 수 있음
- **이 데이터 관문만 뚫리면** 두뇌·해자가 살아난다. 로직(signals/indicators)은 이미 코드로 심어둠 — 데이터만 꽂으면 작동.
